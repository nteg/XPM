MobileApp (Windows Phone)
1. This app based on Windows Phone WinJS depicting the registration of a push notification channel.
2. Receives the push notification and displays in notification center.


REST based WebApp to support Push Notification
1. This web based rest service receives the push notification channel from the windows phone.
2. Retrieves the access token to authenticate the push notification
3. Sends the push notification


*These apps require access to windows phone store, hence needs windows store account.
